# Pixelattack Boilerplate

A personal small scale boilerplate for Pixelattack projects

## Install

```
$ npm install
```

### Directory

```
Project
├── assets
├── README.md
├── node_modules
├── package.json
├── package-lock.json
├── .gitignore
├── .eslintrc
├── .eslintignore
├── gulpfile.js
└── src/
    ├── scss/
    │   ├── components/
    │   │   └── _components.buttons.scss
    │   ├── settings/
    │   │   ├──_settings.config.scss
    │   │   └──_settings.global.scss
    │   └── main.scss
    ├── js
    │   ├── jquery-3.4.1.min.js
    │   ├── modernizr-3.8.0.min
    │   ├── plugins.js
    │   └── main.js
    ├── images
    ├── fonts
    └── index.html
```

## Development

```
$ npm run dev
```

## Production

```
$ npm run build
```
